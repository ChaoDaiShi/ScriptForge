## 1. 产品定位与商业逻辑

### 1.1 一句话定位

**ScriptForge** 是面向中国网络文学作者及内容制作公司的 AI 剧本创作平台，能将 **3 个章节以上的小说文本** 自动转换为结构化 **YAML 格式剧本初稿**，大幅降低 IP 改编门槛，提升微短剧剧本量产效率。

### 1.2 核心商业逻辑

```
小说文本 ─→ ScriptForge AI引擎 ─→ YAML结构化剧本 ─→ 下游消费（人工拍摄 / AI视频生成）
                 │                        │
                 ▼                        ▼
         订阅/API 收入            生态抽成/数据变现
```

ScriptForge 的核心商业价值不在于"AI 写剧本"本身，而在于它通过 **YAML Schema 标准化**卡住了"小说→视频"全链路中最关键的中间管道位置。

### 1.3 市场天板

| 指标                             | 数据                    | 来源           |
| -------------------------------- | ----------------------- | -------------- |
| IP 改编市场规模（2024）          | ¥2,985.6 亿             | 中国社会科学院 |
| 微短剧市场规模（2024）           | ¥500 亿+                | 行业统计       |
| 网文作者规模                     | 3,119.8 万人            | 行业统计       |
| 网文 IP 改编短剧新增授权（2024） | 1,600+ 部，同比增长 81% | 行业统计       |
| 海外网文出海市场                 | ¥50 亿+，用户 3.5 亿    | 行业统计       |

## 3. 五个增量盈利方向

### 3.1 IP 评估报告 + 版权撮合 ⭐（最高优先级）

#### 市场逻辑

网文平台（阅文、番茄、七猫）拥有数百万部小说，但编辑资源极其有限，人工筛选改编 IP 效率极低。ScriptForge 的 AI 引擎可以批量"扫文"，自动输出结构化的改编潜力评估。

#### 产品形态

- **输入**：小说全文（或 50 章以上样本）
- **输出**：一份完整的 IP 评估报告，包含：
  - 改编潜力综合评分（百分制）
  - 视觉化指数（场景丰富度、动作密度、特效需求评估）
  - 爆款概率预测（基于历史爆款数据的对比分析）
  - 类型匹配建议（短剧/长剧/动漫/电影）
  - 目标受众画像
  - 同类作品竞品对标
  - 改编难点与建议

#### 定价策略

| 类型             | 定价         | 说明                                   |
| ---------------- | ------------ | -------------------------------------- |
| **单份评估报告** | ¥299~999/份  | 比人工评估（¥3,000~5,000）便宜 10 倍   |
| **批量评估包**   | ¥9,999/50 份 | 适合 IP 孵化机构批量筛选               |
| **月度榜单订阅** | ¥4,999/月    | "本月最具改编潜力的 50 部网文"定期报告 |

#### 收入测算

假设一家影视公司每周评估 20 部作品，按中等定价 ¥500/份：

- 单客户年收入：¥500 × 20 × 50 周 = **¥50 万/年**
- 目标 10 家 B 端客户：**¥500 万/年**

#### 竞争优势

- 人工评估一部小说需要 1~3 天 × 专业编剧，成本 ¥1,000~3,000
- AI 评估仅需 5~10 分钟，边际成本趋近于零
- 评估标准统一，避免人工评估的主观偏差

### 3.2 YAML → AI 视频工作流抽成（生态税）

#### 市场逻辑

AI 视频生成（可灵、Sora、Pika、Runway、即梦）是未来 3 年最大的 AI ToC 风口。但所有视频工具都面临同一个问题：**缺乏高质量的剧本输入**。ScriptForge 的 YAML Schema 本身就是结构化的"视频指令集"，天生适合被视频生成工具消费。

#### 盈利模式

| 模式             | 说明                                        | 预期抽成比例 |
| ---------------- | ------------------------------------------- | ------------ |
| **API 转接费**   | 用户通过 ScriptForge 一键推送到视频生成平台 | 5%~10%       |
| **联合会员**     | ScriptForge Pro + 视频生成工具额度打包      | ¥299/月      |
| **分镜脚本附加** | 生成视频可直接使用的分镜指令集              | ¥49/次       |

#### 实施路径

1. **M1~M3**：与 1~2 家国内视频平台（可灵、即梦）技术对接，验证 YAML 直转视频的可行性
2. **M4~M6**：推出联合会员产品，验证付费意愿
3. **M7~M12**：扩展至 3~5 家平台，形成网络效应

#### 护城河

一旦制作公司的工作流深度依赖"ScriptForge YAML → 视频工具"的链路，迁移成本会很高。这是**先发优势 + 数据网络效应**的双重壁垒。

### 3.3 剧本交易市场（双边网络效应）

#### 市场逻辑

C 端作者用 ScriptForge 生成剧本后，除了自己用，还可以上架出售；B 端制作公司可以直接在平台上采购优质剧本。

#### 产品形态

- **剧本市集**：作者上架剧本，标注类型、集数、改编难度、售价
- **版权协议模板**：平台提供标准化的授权协议（独家/非独家/分成）
- **评分与推荐**：基于成交数据的剧本热度排名

#### 盈利模式

| 模式         | 说明                                       | 预期费率       |
| ------------ | ------------------------------------------ | -------------- |
| **交易抽成** | 每笔交易的平台服务费                       | 10%~15%        |
| **置顶推荐** | 剧本在搜索结果中的置顶曝光                 | ¥999/周        |
| **独家买断** | ScriptForge 自己买入优质剧本，转售给制作方 | 利润差 20%~50% |

#### 定价参考

微短剧剧本（30 集完整版）当前市场行情：

- 普通剧本：¥5,000~10,000
- 优质剧本（专业编剧）：¥15,000~30,000
- 头部剧本（有数据验证）：¥50,000+

#### 收入测算

- 平台在售剧本：1,000 部
- 月交易量：200 单
- 平均客单价：¥8,000
- 抽成 12%：**¥19.2 万/月**

#### 为什么现在做

微短剧剧本是目前版权交易的蓝海。传统影视剧本交易市场效率极低（信息不对称、交易成本高），而 AI 生成的低成本使供给侧有了规模化放量的可能。

### 3.4 出海多语种剧本工厂（高客单价蓝海）

#### 市场逻辑

中国网文出海 2024 年市场规模超 ¥50 亿，海外用户超 3.5 亿。以 ReelShort（中文在线旗下）为代表的短剧出海平台，在欧美靠霸总短剧月流水超 800 万美元。但海外制作公司面临的核心痛点是：**没有好的剧本来源**。

ScriptForge 的差异化优势是：一次转换出中英两个版本的 YAML 剧本，海外制作公司可以直接拿到英文版拍摄。

#### 产品形态

- **中文→英文剧本转换**：保留剧本结构，适配英文表达习惯
- **双语 YAML**：同一剧本携带 `zh-CN` / `en` 双语文本
- **本地化适配**：中文内心独白多 → 英文更依赖对白和动作，非简单翻译

#### 定价策略

| 套餐           | 价格    | 说明                    |
| -------------- | ------- | ----------------------- |
| **海外创作版** | $49/月  | 30 次转换，含英文输出   |
| **海外专业版** | $199/月 | 120 次转换 + 多语种支持 |
| **企业版**     | $999/月 | 无限使用 + 专属 SLA     |

> **注意**：美元定价与国内 ¥ 定价不在一个量级，海外版收入天花板显著更高。

#### 目标市场

1. **东南亚市场**（印尼、泰国、越南）：短剧增长最快，文化接受度高
2. **日韩市场**：付费意愿强，IP 改编需求大
3. **欧美市场**：客单价高，ReelShort 已验证霸总短剧的需求

### 3.5 行业垂直模型微调服务（企业级 SaaS 天花板）

#### 市场逻辑

头部短剧公司有自己的"风格偏好"——有的喜欢快节奏反转型，有的喜欢甜宠型，有的偏现实主义。通用 AI 模型无法满足这些个性化需求。

#### 服务内容

| 服务项           | 说明                                     | 定价                         |
| ---------------- | ---------------------------------------- | ---------------------------- |
| **专属模型微调** | 用该公司历史剧本来微调专属 LLM           | ¥50,000~200,000（一次性）    |
| **白标定制**     | ScriptForge 变他们的品牌，部署到客户环境 | ¥19,999/月                   |
| **私有化部署**   | 全栈部署到客户自有服务器/VPC             | ¥99,999（一次性）+ ¥9,999/月 |

#### 目标客户

前 10~20 家头部短剧制作公司，年产能 50 部以上的机构。

## 6. 风险识别与对策

### 6.1 风险矩阵

| 风险                        | 概率 | 影响 | 等级    | 对策                                                             |
| --------------------------- | ---- | ---- | ------- | ---------------------------------------------------------------- |
| **LLM API 成本侵蚀毛利**    | 高   | 高   | 🔴 致命 | 自建 prompt caching 层；批量请求折扣谈判；适度上调 B 端 API 定价 |
| **剧本质量天花板**          | 中   | 高   | 🟠 严重 | 从"60 分到 80 分"的 AI 精修功能；降低人工打磨成本                |
| **大平台入场竞争**          | 中   | 中   | 🟡 中等 | 差异化在 YAML Schema 和 B 端 API 能力；不做纯 C 端写作工具       |
| **合规与版权风险**          | 中   | 高   | 🟠 严重 | 上传时版权声明；敏感内容检测；法律条款前置；版权纠纷保险         |
| **AI 视频生成普及不及预期** | 低   | 中   | 🟢 可控 | 主营业务（剧本转换）独立于视频生成；手动拍摄场景也是大市场       |
| **获客成本过高**            | 中   | 中   | 🟡 中等 | 优先做 B 端大客户（低 CAC），C 端靠口碑传播和内容营销            |

### 6.2 成本敏感度分析

| 成本项                       | 占比估计 | 优化方向                                      |
| ---------------------------- | -------- | --------------------------------------------- |
| **LLM API 调用**             | 40%~50%  | 缓存、批量、模型路由（低成本任务走 DeepSeek） |
| **研发人力**                 | 30%~35%  | 核心团队 + 外包配合                           |
| **服务器（Postgres/Redis）** | 5%~10%   | Supabase 按量付费，随用量增长                 |
| **市场 & 销售**              | 10%~15%  | 内容营销为主，减少付费投放                    |
| **行政 & 其他**              | 5%       | 精简                                          |

### 6.3 单位经济模型

| 指标                   | 计算                        | 数值          |
| ---------------------- | --------------------------- | ------------- |
| **平均客单价（C 端）** | (¥49+¥149)/2                | ¥99/月        |
| **平均客单价（B 端）** | (¥1,999+¥9,999)/2           | ¥5,999/月     |
| **单次转换 API 成本**  | 平均 15K tokens × ¥0.003/K  | ¥0.045        |
| **单次转换毛利**       | ¥0.1（API 收入） - ¥0.045   | ¥0.055（55%） |
| **C 端 CAC**           | 内容营销成本 ÷ 新增用户     | 目标 ≤¥50     |
| **B 端 CAC**           | 销售成本 ÷ 新增客户         | 目标 ≤¥1,000  |
| **C 端 LTV**           | ¥99 × 12 月 × 65% 续费率    | ¥772          |
| **B 端 LTV**           | ¥5,999 × 18 月 × 85% 续费率 | ¥91,785       |
| **LTV/CAC（C 端）**    | ¥772 / ¥50                  | **15.4x** ✅  |
| **LTV/CAC（B 端）**    | ¥91,785 / ¥1,000            | **91.8x** ✅  |

> **结论**：B 端客户是盈利核心，单位经济模型非常健康。C 端作为漏斗上层和生态基础，需要控制获客成本。

## 7. 投资人视角结论

### 7.1 核心判断

**ScriptForge 是目前结构最完整、变现路径最清晰的 AI 垂直工具之一。**

它的核心优势不是"AI 写剧本"这个功能本身，而是通过 **YAML Schema** 卡住了"小说→视频"这条未来 3 年最有价值的中间管道。

### 7.2 立即执行的三件事

| 优先级 | 行动                                      | 理由                                     |
| ------ | ----------------------------------------- | ---------------------------------------- |
| **P0** | **M1 先跑通 B 端 API 付费闭环，验证 PMF** | B 端付费意愿最明确，单位经济最好         |
| **P1** | **M3 启动 IP 评估报告产品**               | 零边际成本、高客单价、与核心能力天然匹配 |
| **P2** | **M6 前敲定 1~2 家视频平台合作**          | 打开视频转接费和联合会员的收入天花板     |

### 7.3 一句话总结

> **ScriptForge 不只是卖"AI 转剧本"的工具，而是在构建"小说 → 剧本 → 视频"全链路的行业标准基础设施。订阅收入保底，评估报告和交易抽成放大利润，视频生态抽成决定天花板。B 端大客户是盈利核心，C 端作为生态孵化和品牌阵地。**
